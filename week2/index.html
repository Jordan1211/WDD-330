    Variables:
  • const - is used if variable will not be reassigned later.
  • let - is used if variable may be reassigned later.

To get the value of a variable, simply enter it in the console and press enter. You can also use "typeof" to return the data type.

Scobe refers to where a constant or variable is accessible by the program.

Template Literals use the backtick character ` to deliminate the string. This allows both quote types to be used as well as allows interpolation of JavaScript code.

For example:
'Hello!';
or
const name = `Siri`;
`Hello ${ name }!`;
Result: `Hello Siri!`

Remember Equality is using "=", soft equality is using "==" and hard equality is uinsg "===". Remember to use .isNaN rather than hard equality with the NaN value.

Novice to Ninja Chapter 3 - Arrays, Logic, & Loops

Arrays - an array is an ordered list of values. Arrays are shown in square brackets []

You can add values to Arrays one by one by setting an Array postion equal to a value. More often Array Literals are used as you can define multiple values.

    Arrays have many Properties and Methods:
  • .length - returns the length of an array
  • .pop() - removes the last item from an array
  • .shift() - removes the first item from an array
  • .push() - appends a new value to the end of an array
  • .unshift() - appends a new item to the beginning of an array
  • .concat() - can be used to merge an array with one or more arrays
  • .join() - can be used to turn the array into a string of the items seperated by a comma
  • .slice() - creates a subarray
  • .splice() - removes an item from an array then inserts new items in their place
  • .reverse() - reverses the order of an array
  • .sort() - sorts an array alphabetically - use a callback for numbers
  • .indexOf() - returns the first occurence of the value of an array. Will return -1 if the value is not in the array. Alterntively use .includes()

Sets are a data structure that represents a collection of unique values. They have differing methods than arrays such as .add(), .delete(), .has() and .clear()

Maps are another data structure that are a convenient way of keeping a list of key and value pairs. (Similar to hashes)

Logical conditions allow you to control the flow of a program by running different blocks of code.

Think if else statements and the mathematical operators (+,-,*,/,%,==).

Switch can be used to string lots of if/else statements together to make a logical decision tree

Loops repeat a piece of code over and ober again according to certain conditions.

While - while (condition) {// do somethig}

Do While - do { do something} while(condition)

For - for (initialization ; condition ; after) { do something }

Nested For Loops - you can place a loop inside another loop

Novice to Ninja Chapter 4 - Functions

Functions are a chunk of code that are given a name then that name can be referenced.

To define a function use the "function" declaration followed by the name of the function followed by () and {}. "()" will contain parameters/arguments and "{}" will contain the code for the function.

Functions can also be anonymous when being assigned to variables.

To invoke a function simply enter its name followed by parentheses.

    Arrow Functions
  • Arrow functions can be identified by the arrow symbol => the parameters come before the arrow and the main body of the function comes after. They are always anonymous.
  • They are much less verbose than normal function declarations.
  • Single parameters don't need putting into parentheses.
  • The body of the function doesn't need placing inside a block if it's only one line.
  • The return keyword isn't required if the return statement is the only statement in the body of the function.
  • They don't bind their own value of this to the function.

The JavaScript interpreter moves all variable and function declarations to the top of the current scope regardless of where they are defined. This is known as Hoisting. Variables may be moved to the top of the current scope but their assignment is not hoisted. It won't be assigned until defined.

Callbacks - a function that is passed as an argument to another function.
