Novice to Ninja Chapter 11 - Further Functions

Functions are first-class objects which means they can be passed around in the same way as every other value.

  • They have their own properties and methods

    • One example of a property is the length property that returns the number of parameters the function has.

    call() and apply() are examples of methods. The call() method can be used to set the value of "this" inside a function to an object that is provided as the first argument. (I think it is easier just to use this but in some situations call can be used). The apply() method works the same except the arguments of the function are provided as an array. Which can be useful if the data you're using is already in an array.
    For example:
    square.apply(null, [4])

    You can make your own custom property
    For example:
    square.description = 'Squares a number that is provided as an argument'
    -- 'Squares a number that is provided as an argument'

  • They accept other functions as parameters
  • They can be returned by other functions

    • Immediately Invoked Function Expression – or IIFE – (pronounced 'iffy') is an anonymous function that, as the name suggests, is invoked as soon as it’s defined. This is easily achieved by placing parentheses at the end of the function definition (remember we use parentheses to invoke a function). The function also has to be made into an expression, which is done by placing the whole declaration inside parentheses. IIFE can be leveraged in order to create temporary variables. It can also be used to set up any initialization code that there’ll be no need for again.
